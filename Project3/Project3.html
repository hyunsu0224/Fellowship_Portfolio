清海鎮

概要 : ウェブページを利用し、 日本と韓国に生息している 魚に興味をもたらせる教育用ゲーム
開発期間 : 2019.09.23 ~ 2019.11.15
役割
私 PM, HTML, CSS, JS, Jquery
友達1 SQL(DB), Java(JSP)
友達2 SQL(DB), Java(JSP)
友達3 HTML, CSS, JS, JQUERY
目的
魚に対する興味を上げるため
幼児教育用の魚ゲーム
使用した技術 HTML, CSS, JavaScript, Jquery,Java, SQL
Overview
  • 1. 周波数センサーで数値を測定
  • センサーが敏感すぎて、いびきの音を区別することが大変でした。
  • 解決方　:　同じいびきの周波数を繰り返して測定し、平均の周波数を区別しました。
  • 2. サイズ測定
  • モーターのサイズを間違えて思ったより結果が大きく完成した。
  • 解決方　:　サイズの測定の重要性を傷ついた。 そして、買う前によく調べて買わなければならないということを分かるようになりました。
